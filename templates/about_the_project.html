Инструкция

  • Для использовани всех возможностей: сохранение, просмотр, коррекции - нужно зарегистрироваться в системе
  • Чтоб создать и сохранить новое расписание, зайдите в систему и нажмить кнопку "Добавить новое расписание на день"
    • В открывшемся окне необходимо нажать на одну из трёх кнопок "Добавить ячейку", чтоб добавить выбор в соответствующий столбец
    • Далее выбираем желаемый продукт из списка, с возможностью поиска
    • В нижней части окна можно быстро посмотреть общую статистику (обновляется после любого действия на странице)
  • Так же можно расчитать витамин К без регистрации, нажав на "Расчитать без регистрации", но без сохранения результата (работает аналогично)

Рекомендации для использования

  • Прежде чем пользоваться этим калькулятором обязательно проконсультируйтесь с вашим врачом
  • Использовать приложение лучше всего на компьютере
  • Использовать браузер "Chrome"
